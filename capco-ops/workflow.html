💡 Integration Notes

  • SMS: Twilio ($0.0079/msg) — no Linqto needed. Simple API, works immediately.
  • Magic Links: Hosted on your existing Railway/Astro stack. Auth via unique token in URL (expires 7 days).
  • GoHighLevel: API for pipeline stages, contact lookup, opportunity tracking.
  • Email: Already have IMAP on agent@capcofire.com. Can add jk@ if needed.
  • AI: Classification + drafting via Claude API (Sonnet for cost efficiency).
  • Project DB: Supabase — already have it. New tables for tasks, projects, magic links.

PROCESS

🎯 Client Experience Philosophy

  • Zero friction: No accounts, no passwords, no portals. Tap link → do the thing → done.
  • One-pagers only: Client never sees a dashboard. Each link = one focused task.
  • Smart pre-fill: If we know the address, occupancy type, etc. — it's already there.
  • Mobile-first: These pages must work perfectly on a phone. Contractors are on-site, not at desks.
  • Upload = done: No confirmation emails to reply to, no "we'll get back to you." Instant feedback.
  • Twilio SMS cost: ~$0.0079/message sent, ~$0.0075/received. A phone number is $1/month. Trivial.
